# StuHDySpaces

## Steps

1. Clone the repo.
2. Run `npm install` in the project directory
3. Run the app

## Run the app

`npm run dev` in `client/`.

## Backend 

This app now also uses a backend. Initialise it by running

`npm run start` in `server/`
